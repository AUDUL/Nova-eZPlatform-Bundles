# NovaeZSolrSearchExtrasBundle

 A bundle which extend ezplatform solr search handler.
 
## Features

- [WIP] Binary file indexation in full text field
- [WIP] Boost doc based on publish date (newer docs score higher)
- [WIP] Highlighting
